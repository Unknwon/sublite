SubLite
=======

SubLite is a theme converter from Sublime Text to LiteIDE.

- Installation:

		go get github.com/Unknwon/sublite

- Convert theme:

		sublite path/to/sublime/theme

Converted file will be in the same folder with suffix `.xml`.
